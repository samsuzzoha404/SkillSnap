Localization:

6. AI MATCHING ENGINE
==================================================

Do NOT fake the AI with marketing words only.
Implement a real explainable matching engine for MVP.

Create a backend matching service that ranks providers using weighted scoring based on:
- distance from consumer
- provider average rating
- provider current availability
- provider workload / active jobs
- provider acceptance rate
- provider completion rate
- category match
- service radius coverage
- past performance / reputation score

- implement a deterministic scoring engine with adjustable weights in config
- return ranked providers with score breakdown
- store score explanation for debugging/admin review

Optional bonus:
- dynamic pricing suggestion based on urgency, demand, distance, and provider base pricing
- fraud/risk flags for suspicious cancellation/payment behavior
